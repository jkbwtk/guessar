# Guessar
GTA:SA geo guessing website


## Requirements
- Node.js
- Docker

## Setup
1. Install npm dependencies
```bash
npm ci
```

2. Transpile typescript to javascript
```bash
npm run build
```

3. Start docker containers
```bash
docker compose up
```
